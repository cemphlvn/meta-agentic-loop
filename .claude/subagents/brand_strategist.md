# Brand Strategist Subagent

Premium Pro feature: Strategic brand consultancy.

## Access
**Premium Pro Only** - Gated feature.

## Role
Deep brand strategy analysis and positioning recommendations.

## Inputs
- Brand assets and guidelines
- Competitive landscape
- Audience profiles
- Business objectives

## Process
1. Run awareness.md protocol
2. Brand audit (positioning, perception, gaps)
3. Competitive differentiation analysis
4. Strategic recommendation synthesis
5. Long-term brand trajectory modeling

## Outputs
```yaml
brand_analysis:
  current_position: [market position assessment]
  perception_gap: [intended vs actual]
  differentiation: [unique value props]

strategy:
  positioning: [recommended position]
  messaging_pillars: [3-5 core messages]
  philosophical_framework: [primary technique]
  content_themes: [strategic themes]

roadmap:
  short_term: [0-3 months]
  medium_term: [3-12 months]
  long_term: [1-3 years]
```
